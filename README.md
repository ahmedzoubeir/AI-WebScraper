# AI-Powered Web Scraper

## Overview
This project is an AI-powered web scraper that utilizes LangChain and Ollama for intelligent DOM parsing and data extraction. It enables users to extract relevant information from web pages using natural language descriptions.

## Features
- **Natural Language Processing (NLP):** Leverages AI to understand user queries in natural language and perform web scraping accordingly.
- **DOM Parsing:** Intelligent parsing of web page structures to locate and extract data efficiently.
- **Extensibility:** Easily adaptable to various websites and data formats with minimal configuration.

## Getting Started
1. **Clone the Repository:**
   ```bash
   git clone https://github.com/ahmedzoubeir/AI-WebScraper.git
   cd AI-WebScraper
   ```

2. **Install Dependencies:**
   Ensure you have Python and the necessary libraries installed. You can install the required packages using pip:
   ```bash
   pip install -r requirements.txt
   ```

3. **Configure the Scraper:**
   Edit the configuration file to specify the target website and the data points you wish to extract.

4. **Run the Scraper:**
   Execute the scraper using the following command:
   ```bash
   python scraper.py
   ```

## Usage
To use the web scraper, provide a natural language description of the data you want to extract. The AI will parse the webpage's DOM and return the requested information.

Example:
```plaintext
Extract the product names and prices from the webpage.
```
